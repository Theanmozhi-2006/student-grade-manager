# Student Grade Manager

A simple Python command-line application to manage student grades.

## Features
- Add students and marks
- View all student records
- Find highest and lowest scorer
- Calculate class average
- Interactive menu

## Technologies Used
- Python 3

## How to Run
python student_grade_manager.py
